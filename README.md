# Face Detection Attendace

### This project is simple attendance detection for empoyee using face detection
<br>face comparison using distance of 2 images from <b>face_recognition</b> module then using pyqt5 for the gui.
<br>for register image save locally in folder Image Storage.
<br>employee data save in <b>.csv</b> format

<br><b>Modules</b>
<ul>
  <li>face_recognition</li>
  <li>pyqt5</li>
  <li>opencv-python</li>
  <li>cmake</li>
  <li>pandas</li>
</ul>

### Instalation Modules
```bash
pip install -r requirements.txt
```

### Run Register New Employee
```bash
python register.py
```

### Run Attendance Detection
```bash
python detect.py
```
